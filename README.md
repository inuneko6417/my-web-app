# RecipeTube 

YouTubeのレシピ動画URLから手順を自動抽出できるWebアプリ。

**URL**: https://recipe-tube-azure.vercel.app/

---

## 概要

RecipeTubeは、YouTubeのレシピ動画URLを入力するだけで、AIが材料・分量を自動で書き起こすWebアプリケーションです。
料理中にレシピをテキストで確認できます。

### 主な機能

- YouTube動画URLを入力するだけでレシピを自動抽出
- AIによる材料・分量の解析・表示
- 登録不要ですぐに使える

## 追加予定機能

| 機能 | 説明 |
|------|------|
| レシピ一覧表示 | 抽出したレシピを一覧で確認 |
| レシピ検索 | キーワードによるレシピの検索 |
| 会員登録・ログイン | ユーザー認証機能 |
| お気に入り登録 | 気に入ったレシピをお気に入りに追加 |
| お気に入り一覧 | 自分がお気に入り登録したレシピを一覧表示 |
| 代替食材提案 | 食材が手元にない場合に代替食材をAIが提案 |
| カレンダー| カレンダー化し週ごとにどれぐらい食材必要か視える化 |
